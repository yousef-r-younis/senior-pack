The Understanding Test

Before using ANY AI-generated content (code or text), ask yourself these questions:

‚úÖ You Pass If You Can:

  • Explain the code line by line to your advisor
  • Modify the code if requirements change
  • Debug it when something breaks
  • Justify why this approach was chosen
  • Write something similar without AI next time
  • Answer "why?" questions about design decisions

‚ùå Red Flags (Learn More First):

  • "I don't know what this line does"
  • "It works, but I'm not sure why"
  • "AI said to do it this way"
  • "I'd need to ask AI to modify it"
  • "I just copied the suggestion"
  • "I can't explain the algorithm"

üí° What to Do When You Don't Pass

  • Ask the AI to explain the code step by step
  • Try to rewrite it yourself from understanding
  • Ask the AI to quiz you on the concept
  • Look up the official documentation for the technology
  • Discuss with teammates or advisor until you understand
